# SonarQube Code Quality Configuration

To run SonarQube analysis for the Deep Skilling projects:
1. Start SonarQube server locally (default port 9000).
2. Ensure SonarScanner is installed and added to PATH.
3. Run `sonar-scanner` in the project root directory containing `sonar-project.properties`.
4. Review the generated code quality report at `http://localhost:9000`.
