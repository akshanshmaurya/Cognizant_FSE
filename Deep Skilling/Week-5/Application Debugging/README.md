# Application Debugging Demo

This module demonstrates how to use the IDE debugger to identify and fix common issues in a Java application.

1. **BuggyCalculator.java**: Contains methods that may throw exceptions or produce incorrect results.
2. Instructions are provided as comments within the code to guide where to place breakpoints.
3. Use your IDE (Eclipse/IntelliJ/VS Code) to run the `main` method in debug mode.
4. Use stepping commands (Step Over, Step Into) to trace execution and inspect variables.
